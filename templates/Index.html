Document Repository

Securely store, organize, and access project-related documents all in one place.

Supporting Tools

Access multiple modules to upload data, generate insights, view history, and download reports.

Value-Set, Roles, and Rules

Truncate or update the value-set collection based on OrgCode and ENV. This requires passing the relevant Excel data.

Risk Score

Fetch records from the risk-score-logs collection based on assester ID and ENV. Optionally, you can filter the results using a start date and end date if a date range is required.

Permission and Roles Update

Update the modelPermission field in existing documents within the permissions and roles collections, excluding those associated with the SuperAdmin role.

Patient Details

Fetch user details across all collections using a mobile number based on the given ENV. Additionally, export the fetched response as an Excel file.

Questionnaire

Create questions in the question collection based on the provided asserterIds, orgDetails, facDetails, and roleDetails.
